is_param - Check if a parameter is present in a parameter list

Calling Sequence

[res,err] = is_param(list_name,param_name)

Parameters

  • list_name : the list of parameters. This list must have been initialize by a call to init_param.
  • param_name : a string. The name of the parameter to be add in the list of parameters.
  • res : the result: %T is the parameter is present, %F otherwise.
  • err : an error flag which is set to %T if list_name is not of type plist (this list hasn't been initialized by a call to init_param).

Description

This function checks if a parameter is present in a parameter list.

Examples 

   mylist = init_param();
   mylist = add_param(mylist,'minbound',[0 0 0]);
   disp(is_param(mylist,'minbound'));
   disp(is_param(mylist,'maxbound'));
