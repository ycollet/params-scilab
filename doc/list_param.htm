list_param - List all the parameters name in a list of parameters

Calling Sequence

[string_list,err] = list_param(list_name)

Parameters

  • list_name : the list of parameters. This list must have been initialize by a call to init_param.
  • string_list : the list of parameters name.
  • err : an error flag which is set to %T if list_name is not of type plist (this list hasn't been initialized by a call to init_param).

Description

List all the parameters name in a list of parameters.

Examples 

   mylist = init_param();
   mylist = add_param(mylist,'minbound',[0 0 0]);
   mylist = add_param(mylist,'maxbound',[1 1 1]);
   disp(list_param(mylist));
